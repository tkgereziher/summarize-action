Welcome to Summarize & Action

Thanks for installing! A quick guide to get started:

  1. Pin the extension to your browser toolbar so it's always visible.
  2. Select text on any webpage, click the extension icon (top-right), and it will summarize the selection and suggest actions.
  3. Use the context menu: right-click selected text and choose "Summarize selection..."

We send selected text to the local server / your configured LLM endpoint. See the README for privacy details.
